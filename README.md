Study & Task Tracker

A Python desktop application built using Tkinter to manage study tasks 
with time-based reminders.

Features:
- Add, update, and delete study tasks
- Time-based reminders and notifications
- Multithreading for background reminder checks
- JSON-based persistent task storage
- Simple and user-friendly GUI

Technologies Used:
- Python
- Tkinter
- Multithreading
- JSON

How to Run:
1. Install Python 3.x
2. Run the application using:
   python Study&TaskTracker.py
